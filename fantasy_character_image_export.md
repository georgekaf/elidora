## 1. Archmage Arcturus — Character Builder / Turnaround Concept

### Core Character Direction

**Character:** Archmage Arcturus  
**Role:** Elderly archmage connected to the Nexus / Nexus Pillars concept  
**Format:** Character-builder concept sheet  
**Style:** Semi-realistic digital fantasy character concept art, painterly rendering  
**View:** Same character shown from the front and back  
**Background:** Isolated or minimal; transparent preferred when usable  

### Required Design Notes

- Old human archmage.
- Balding head.
- White beard.
- Serious, stern, weathered face.
- Robes and cloak, not armor.
- Staff, not swords.
- Full body visible, including feet.
- No text in the image unless specifically requested.
- Front and back views should clearly be the same character.
- Nexus-related motifs should appear through subtle arcane details, staff crystal, robe trims, glyphs, or faint pillar influence.

## 4. Nexus Character Creator Screen

### Core Concept

A fantasy RPG-style character creation UI where a Nexus mage stands on the left and editable character-building options appear on the right.

### Required UI Elements

- Large full-body Nexus mage on the left.
- Faint Nexus Pillars in the background.
- UI panels on the right.
- Tabs should include:
  - Appearance
  - Pillars
  - Equipment
  - Spells
- Sliders for:
  - Body
  - Mind
  - Soul
- Small portrait presets.
- Bottom button should read exactly:
  - **AWAKEN NEXUS HERO**
- Text above the button:
  - Label: **Character name**
  - User input: **Twinkie**

### Prompt — Nexus Character Creator UI

```text
semi-realistic digital fantasy illustration of a game character creator screen, large full-body Nexus mage on the left, faint Nexus Pillars in the background, dark ornate fantasy UI panels on the right, small portrait presets at the top, tabs labeled Appearance, Pillars, Equipment, Spells, sliders labeled Body, Mind, Soul, a text field above the bottom button labeled Character name with the user input Twinkie, big bottom button reading AWAKEN NEXUS HERO, painterly rendering, high detail, concept art, fantasy RPG interface
```

### Notes From Iteration

- Avoid duplicate tab labels such as “Pillars” appearing twice.
- Keep the bottom button text as **AWAKEN NEXUS HERO**.
- The character name section should show **Character name** as the label and **Twinkie** as the input value.

## 7. Style Guide Summary

Use this style language when trying to keep everything consistent:

```text
semi-realistic digital fantasy illustration, painterly rendering, Dungeons and Dragons character concept art, detailed cloth and armor, realistic materials, fantasy RPG design, full-body character, isolated character, transparent background, high detail
```

For turnarounds:

```text
front and back turnaround of the same character, full-body, feet fully visible, consistent outfit and proportions, character concept sheet, no text
```

For transparent/cut-out characters:

```text
isolated character, transparent background, no scenery, no ground, no cast shadow, clean cut-out edges
```
